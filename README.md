## Running Locally

Clone the project

```bash
git clone https://github.com/s0phL/somebodys-random-rooms.git
```

Go to the project directory

```bash
cd somebodys-random-rooms
```

Install dependencies

```bash
npm install
```

```shell
├── firebase-admin@13.6.1
├── firebase@12.9.0
├── phaser@3.90.0
└── vite@7.3.1
```

Start the development server

```bash
npm run dev
```
